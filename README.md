## Key Mapping
### Terminal
- Opening: <kbd>Ctrl+t</kbd>

### File explorer
- Opening: <kbd>\\+b</kbd> or <kbd>Leader+b</kbd>
- Closing: <kbd>\\+B</kbd> or <kbd>Leader+B</kbd>

### Spliting
- Vertical Spliting: <kbd>\\+vs</kbd> or <kbd>Leader+vs</kbd>

### Live grep
- Opening: <kbd>\\+lg</kbd> or <kbd>Leader+lg</kbd>
